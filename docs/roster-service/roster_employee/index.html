Add Roster Employee#

Add a Roster Employee that belongs to a Roster and Employee to the database with basic information

URL/api/rosters/{rosterId}/employees/{employeeId}
MethodPOST
Auth requiredYes
Permissions requiredROLE_EMPLOYER

Data constraints

{  "fromDateTime": "YYYY-MM-DDThh:mm:ss.sss",  "toDateTime": "YYYY-MM-DDThh:mm:ss.sss"}

Success Response#

Code : 200 OK

Content examples

Response will reflect back a representation of the newly added Roster Employee.

For a Roster Employee with ID 243fbb89-7fba-4e8f-a25f-44d439d6a0f0 and for the date of 2021-09-16 on the local database.

Note that id is auto-generated and of UUID datatype

{  "id": "243fbb89-7fba-4e8f-a25f-44d439d6a0f0",  "roster": {    "id": "bff4ccd3-bae4-4c2f-a8bd-fdc703df8c25",    "date": "2021-09-16"  },  "employee": {    "id": "a88f8825-34e2-44cb-a9f5-4482c82bbc48",    "name": "Employee 1",    "vaccinationStatus": "SECOND_DOSE",    "vaccinationBrand": "PFIZER",    "healthStatus": "HEALTHY"  },  "fromDateTime": "2022-09-16T15:00:00",  "toDateTime": "2022-09-16T16:00:00",  "currentHealthStatus": "HEALTHY"}

Error Response#

Code : 400 Bad Request

Content examples

Response will reflect back an error when the Request Body does not conform to the constraints.

For a Roster with invalid Date format
